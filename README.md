# O CAMINHO DA FORÇA
***
## Introdução
"O CAMINHO DA FORÇA" é um projeto acadêmico desenvolvido com o objetivo de integrar a nota da matéria de Pesquisa e Inovação, como sendo o Projeto Induvidual, desenvolvido pelo aluno Gustavo Vieira Santana, estudante da SPTech (São Paulo Tech School) no primeiro semestre de ADS (Análise e Desenvolvimento de Sistemas) turma B.
***

## Sobre o projeto

O projeto é tem o objetivo de introduzir o usuário ao tema "Força" e ajudá-lo a enfrentar os maiores vícios da modernidade, como o vício em redes sociais, jogos e pornografia, insigar o usuário a refletir sobre seu estado mental e bloqueios, bem como entregar diagnósticos baseados em suas respostas, e entregar uma ação simples para que o usuário possa se libertar desses vícios e alcançar a força interior, mostrando ao final que esta força vêm de ter um propósito.
***

## Features
- Login e cadastro de usuário
- Dashboard inicial com estágio, nível de força e barra de progresso
- Checkin diário com bônus de XP para incentivar o usuário a voltar diariamente
- Tela com cinco demônios representando os vícios da modernidade, cada um com botões para ações específicas (cada ação retorna uma quantia de XP diferente)
- Checkin diário para o usuário refletir sobre seu estado mental e bloqueios
- Diagnóstico baseado nas respostas do checkin diário
- Dashboard mostrando gráfico de bloqueios semanais, o resultado do diagnóstico e uma ação simples para o usuário alcançar a força interior
- Sistema de níveis e XP para gamificar a experiência do usuário
***
## Tecnologias utilizadas
- Frontend: HTML, CSS, JavaScript, Chart.js
- Backend: Node.js, Express, MySQL
***

## Como testar

- Instale as dependências
  1. `cd backend/`
  2. `npm intall -y`
  3. `cd ..`
  4. `cd frontend/`
  5. `npm install -y`
- Inicie o servidor frontend e backend
  1. `npm start` rode o comando nas pastas `backend/` e `frontend/`
